activity-logger

local-first work-history tracker

in progressFastAPINext.js 14SQLite

Overview

A full-stack, local-first tool for tracking work activity history over time — built as a deployable monorepo.

Architecture

Backend: FastAPI with SQLAlchemy over SQLite. Frontend: Next.js 14. Packaged as a monorepo so backend and frontend ship and versioned together not separate deploys to coordinate by hand.

A related piece of infrastructure work sits alongside this: a Docker-first full-stack setup (React + FastAPI + PostgreSQL) emphasizing layered service architecture, Alembic migrations, and manual JWT authentication — the same architectural instincts applied to a heavier stack.
