AURA

AI digital clone platform — frontend architecture

in progressReactteam project

Overview

AURA is an AI digital clone platform built with a small team — Rohit and Humam alongside this work. The contribution here is frontend architecture: the interface layer a user actually talks to the clone through, not the model behind it.

Problem

The chat surface needed to hold multiple concurrent conversations without state bleeding between them, stream model output live, accept voice as an input method, and render markdown correctly — while carrying a visual identity that didn't look like a generic chat template.

Architecture & work done

  • Rebuilt Chat.jsx around a multi-session store architecture, replacing whatever single-thread assumptions were baked into the earlier version.
  • Added streaming responses, voice input, and markdown rendering to the chat surface.
  • Added conversation export.
  • Fixed a series of production bugs across multiple work sessions.
  • Led a full rebrand using AURA's actual logo mark, built into a white/amber design system.

Status

In progress, built collaboratively. Not yet public — the live link goes here once it ships.
