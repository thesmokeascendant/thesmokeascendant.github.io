corrupted-csv

Delimiter auto-detection, ragged-row repair, and encoding-error handling.

completePython

Overview

A CSV repair tool: delimiter auto-detection, ragged-row padding/truncation/dropping, null-token normalization, and BOM/encoding-error handling.

Architecture

Uses Python's Sniffer for delimiter detection with a fallback path, plus a header-vs-data heuristic for files with ambiguous first rows.

Engineering decisions

Follows the same recovery-report pattern as project 01 (log-analyzer) — every repair is logged, not silent.

Tradeoffs

Auto-repair trades strict correctness for usability on real-world messy CSVs, where rejecting the file outright is often worse than a documented best-effort fix.

Lessons learned

The recovery-report pattern from the very first project in this series turned out to be reusable as a general shape for any tool that repairs rather than rejects.
