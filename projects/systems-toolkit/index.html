Overview

Ten self-contained tools, built one at a time and fully finished before the next started: a log analyzer, an API client, a web scraper, a PDF parser, a file organizer, a git-health checker, an SSH automation runner, a CSV repair tool, a duplicate-file finder, and a pluggable AI CLI.

Each project ships with a full README (problem, requirements, architecture, design decisions, tradeoffs, edge cases, examples, limitations, lessons learned, references), an engineering journal, and a passing test suite. Nothing here is a stub — 184 tests total, all passing, across the ten.

01 log-analyzer complete

Fault-tolerant streaming log parser with recovery reporting.

10 ai-cli complete

Pluggable LLM backend with prompt templating and persisted conversation history.
